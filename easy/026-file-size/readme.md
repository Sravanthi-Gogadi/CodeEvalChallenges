<h2>File Size</h2>

<h3>Challenge Description:</h3>

<p>
    Print the size of a file in bytes.
</p>

<h3>Input:</h3>
<p>
    The first argument to your program has the path to the file you need to check the size of.
</p>

<h3>Output sample:</h3>

<p>
    Print the size of the file in bytes.
E.g.
</p>

<pre>55</pre>
